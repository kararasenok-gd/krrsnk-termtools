## Установка
1. Обновляемся
```bash
apt update && apt upgrade
```
2. Устанавливаем Git и Python
```bash
pkg install git && pkg install python
```
3. Клонируем
```bash
git clone https://github.com/kararasenok-gd/krrsnk-termtools && cd krrsnk-termtools
```
4. Запускаем
```bash
python main.py
```
